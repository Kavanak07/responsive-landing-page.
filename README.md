# responsive-landing-page.

This project is a simple **responsive landing page** created using **HTML5** and **CSS3** as part of Web Development Internship Task 1.

## Objective
To build a clean landing page containing:

- Header with logo and navigation links  
- Hero section with heading, text, and call-to-action button  
- Footer section  
- Responsive design for mobile and desktop screens  

## Technologies Used
- HTML5  
- CSS3  
- Flexbox  
- Media Queri
